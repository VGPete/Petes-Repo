What I learned

This week I learned a lot about the robust function abilities that JavaScript has. There are a lot of QoL features that other languages dont have. I especially liked the video the book suggested where it taught us about the stack and queue in browsers where I learned to better understand how to use JavaScript to do Asyncronous computing. I got the Quiz Ninja working where its fetching from the server and has a much larger list of heroes to cycle through. The team project was also very interesting as I learned to work with a team on an existing project. I realized that object oriented programming might be more of my strong suit than others on my team. I still learn a lot from each of them when it comes to HTML and CSS though and shorthand JavaScript as well.
